# DRE Creator Assets Dashboard

A creator assets management dashboard built for Digital Realm Entertainment.

## Stack

- React 18 + TypeScript
- Tailwind CSS
- Vite
- Vitest + Testing Library

## Setup

```bash
npm install
npm run dev
```

## Run Tests

```bash
npm test
```

## Build

```bash
npm run build
```

## Features

- Asset list with loading, empty, and error states
- Search assets by name
- Filter by asset type and status
- Asset detail view
- Upload / create asset form with validation
